b## Reactive
* 리액트 생성, 조회, 업데이트, 삭제 예제
* React C.R.U.D(Create, Read, Update, Delete) Template(Feat. Sequelize)

### 프로젝트 실행
> npm run start

### API 서버 실행
> npm start server

## Packages
* 패키지 복원
  >`npm install`

* `typescript`
* `sequelize`
* `axios`
* `mysql2`
* `react`
* `express`

### Optional

* `nodemon`
* `cors`
* `react-router-dom`
